Dictionary

What word would you like the definition of?
 

I found this API in the github list provided to us on the course website.

Website: https://www.wordsapi.com/

I chose to only include the definition option for the word that the user inputs.

This could be useful in a simple dictionary app, or it could be useful in a reading app.
For example, if a user of this app is reading a book on it and does not know what one of the
words mean, they could tap on it and the app could request the definition from this API and
could deisplay it back to the user.
